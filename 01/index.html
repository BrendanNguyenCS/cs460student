RO.ME "3 DREAMS OF BLACK"

RO.ME is an interactive movie created using WebGL that allows the user to control the movement of the first-person view. Since the movie was created in 2011, it was one of the first projects which showcased the potential of WebGL and what can be created on a browser.

Technology: RO.ME uses WebGL to combine videos, 2D graphics, and 3D graphics. In addition to the graphics, the movie also incorporates music. It's incredible to see what can be created by using just HTML and Javascript.
